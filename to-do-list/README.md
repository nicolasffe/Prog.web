# To-Do List

Projeto front-end de uma agenda de tarefas inteligente feito com `HTML`, `CSS` e `JavaScript` puro.

## Estrutura

```text
to-do-list/
|-- assets/
|   |-- css/
|   |   `-- style.css
|   `-- js/
|       `-- script.js
|-- docs/
|   |-- documentacao.html
|   `-- documentacao.pdf
|-- index.html
`-- README.md
```

## O que o projeto faz

- Cadastra tarefas com nome, categoria, prioridade e data limite.
- Exibe as tarefas em cards com destaque visual por prioridade.
- Permite concluir, reabrir e excluir tarefas.
- Filtra por categoria e ordena por prioridade, data ou mais recentes.
- Salva tarefas e tema no `localStorage`.
- Possui modo escuro, contador de tarefas e mensagem de lista vazia.

## Como rodar

Abra o arquivo `index.html` no navegador.

Se quiser usar um servidor local, rode na pasta do projeto:

```powershell
npx serve .
```

Depois abra o endereço informado no terminal.
